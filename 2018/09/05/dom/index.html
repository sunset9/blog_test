DOM, Document Object Model

– 문서 객체 모델
– document 내장객체를 뜻한다
– HTML 문서의 태그
– <body> 태그 내 요소들의 계층구조 == DOM
– HTML문서의 요소(태그)들을 자바스크립트 코드로 접근하고 관리하기 위한 인터페이스

관련 용어

  • 요소(Element)
    : HTML문서 내의 태그

  • 문서 객체(Document Object)
    : 자바스크립트로 요소에 접근할 수 있도록 객체화 한 것
    HTML요소와 자바스크립트의 연결 지점(인터페이스)

  • DOM 트리
    : 문서 객체를 트리구조(계층구조)로 표현한 것

  • 노드, Node
    : DOM 트리 구성 요소

     – 요소 노드: 태그 요소
     – 텍스트 노드: 태그 요소가 감싸고 있는 텍스트
    ex) <h1> Hello </h1>
         : <h1> - 요소 노드, “Hello” - 텍스트 노드


DOM 객체를 사용하는 코드의 적용 위치

  1. <head>태그 내 <script>태그에서 window.onload 의 리스너로 작성

  2. </body> 닫는 태그 바로 위에 <script>태그에서 작성
    (window.onload 필요 없음)

  3. 일반 함수에서 사용하고 해당 함수를 태그의 이벤트 리스너로 작성

** DOM객체는 <body>의 계층구조를 모두 확인하고 DOM트리가 완성된 이후에 사용해야 한다.
** DOM트리가 완료되는 시점까지 DOM객체를 사용하는 코드가 실행되는 것을 미뤄야 한다.
