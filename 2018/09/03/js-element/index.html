2018-09-03

자바스크립트 핵심 요소 4가지

  1. 자바스크립트 Core 문법
    기본 문법
    데이터 타입, 변수, 제어문, 함수, …
    클래스, 객체, …

    내장 객체(Core Library)(built-in)
    String, Date, Math, Array, Number, Object, …

  1. BOM, Browser Object Model, 브라우저 객체 모델
    웹 브라우저 창(윈도우)과 자바스크립트가 통신하기 위한 수단으로 제공되는 객체들

    – window객체

    • 브라우저 전반적인 기능 제어
    • 자바스크립트에서 정의한 모든 전역 변수, 함수, 객체들은 window객체의 멤버로 추가된다
      ( 전역 변수 var num; -> window.num = 1; 으로 사용 : window는 생략 가능해서 그냥 num = 1; 으로 사용하는 것)
    • Javacript 최상위 객체


    – navigator객체

    • 브라우저의 정보와 운영체제의 정보를 제공하는 객체


    – location객체

    • URL과 관련된 인터넷 주소 정보 관련 객체


    – history객체

    • 사용자 방문기록 관련 객체


    – document객체

    • 웹페이지 문서 관련 객체


    –screen객체

    • 사용자 모니터 정보 관련 객체
  1. DOM, Document Object Model, 문서 객체 모델
    (별도 정리)

  2. Event
    (별도 정리)
