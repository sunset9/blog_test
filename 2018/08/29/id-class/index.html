####id, class 속성

  • id 속성 ( id=”idName”, #idName )
    HTML 문서에서 각 요소들을 구분하기 위한 목적으로 사용하는 속성값
    주로 해당 요소에 기능(이벤트)을 부여할 때 사용한다

  • class 속성 ( class=”className1 className2”, .className )
    HTML 문서에서 공통적인 기능(스타일)을 수행해야하는 요소들을 그룹화하여 관리할 때 사용하는 속성

    -> 같은 스타일을 여러 요소에 적용
    -> 같은 이벤트를 여러 요소에 적용

    여러 개의 클래스값을 띄어쓰기로 구분해서 부여할 수 있다

** ID는 웹 브라우저가 참조값을 저장해놔서 적용 속도가 빠르다.
CLASS 는 따로 저장해놓지 않아서 문서를 다시 처음부터 읽으면서 CLASS요소들을 찾고 적용해준다.
그래서 CLASS가 너무 많거나 구조가 복잡해지면 속도가 느려진다.
